# market_monitor

#### 1. 克隆项目
``` shell
$ git clone https://github.com/JiangInk/market_monitor.git
```

#### 2. 下载项目依赖
``` shell
$ cd market_monitor
$ go mod download
```

#### 3. 创建数据库
``` sql
CREATE DATABASE monitor DEFAULT CHARACTER SET utf8 DEFAULT COLLATE utf8_general_ci;
``` 

#### 4. 运行项目
``` shell
$ go run main.go
```
